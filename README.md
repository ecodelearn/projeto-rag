# projeto-rag

Base local para baixar e analisar os projetos de RAG referenciados no livro do Sandeco.

## Estrutura
- `downloads/`: zips baixados do Google Drive
- `sources/`: projetos extraidos
- `scripts/`: utilitarios de download
- `analises/`: anotações de benchmark e comparativos

## Projetos baixados
- `rag_classico` → https://bit.ly/sandeco_rag_classico
- `rag_memory` → https://bit.ly/sandeco-rag-memory
- `agentic_rag` → https://bit.ly/sandeco-agentic-rag
- `crag` → https://bit.ly/sandeco-CRAG
- `adaptative_rag` → https://bit.ly/sandeco-adaptative-rag
- `rag_fusion` → https://bit.ly/sandeco-rag-fusion
- `hyde_rag` → https://bit.ly/sandeco-hyde-rag

## Nota importante
- `hyde_rag` e `rag_fusion` vieram do mesmo arquivo (mesmo ID no Drive), ou seja, conteúdo idêntico.
